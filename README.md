# Kopi Kana — Landing Page

Ini adalah landing page sederhana untuk brand "Kopi Kana".

Cara menjalankan secara lokal:

1. Buka file `index.html` di browser (double click atau open with browser).
2. Untuk fitur form berlangganan, tidak diperlukan server — data disimpan sementara di `localStorage`.

File penting:
- `index.html` — halaman utama
- `css/styles.css` — stylesheet
- `js/script.js` — script kecil untuk form

Ganti nomor WhatsApp di `index.html` pada link WA dengan nomor Anda sebelum digunakan.
